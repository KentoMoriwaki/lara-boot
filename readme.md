## lara-boot

Laravel bootstrap template

## Install

Clone this repository and

```
composer install
npm install
bower install
```

Enjoy it!

## Include

* Laravel 4.1
* Twitter bootstrap 3
* Sass(Compass)
* Grunt
** grunt-contrib-cancat
** grunt-contrib-watch
** grunt-contrib-compass
